Projeto de Testes Automatizados com Cypress
Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de demonstrar habilidades em testes automatizados utilizando o Cypress. Ele simula cenários reais de testes em uma aplicação web, garantindo qualidade e confiabilidade do sistema.

 Tecnologias Utilizadas
JavaScript
Cypress
Node.js
 Estrutura do Projeto

    home.cy.js

  support/

 Como Executar o Projeto
1. Clonar o repositório
git clone https://github.com/seu-usuario/seu-repositorio.git
2. Instalar dependências
npm install
3. Executar o Cypress

Modo interativo:

npx cypress open

Modo headless:

npx cypress run
